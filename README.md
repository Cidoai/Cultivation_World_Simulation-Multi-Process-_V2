# Cultivation_World_Simulation-Multi-Process-_V2
Cultivation_World_Simulation(Multi-Process)_V2

文件里的 build.bat 是用来编译的文件。Windows下运行它后，会生成一个Cultivation_World_Simulation(Multi-Process)_V2.exe，运行它，耐心等待，就可以得到Cultivation_World_Simulation_V2.out和cultivation_stats_V2.csv。

本项目主要由AI实现，各个参数的解释都在代码注释中。

修仙世界模拟实现建议 (1).md 是作者本人和AI的对话，项目主要代码基本出自于此，我主要调了一些参数。

调参可以观察更多的世界模拟方式，也可以适当缩短时间。
